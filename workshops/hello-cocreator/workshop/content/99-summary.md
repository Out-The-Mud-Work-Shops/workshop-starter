# Summary

That's it — you ran a one-page workshop served directly from a GitHub repo.

## Make it yours

1. Copy this folder to `workshops/<your-workshop-name>/`.
2. Edit `resources/workshop.yaml` (title, description) — keep `session` within
   the allowed clamp (see the repo's `CONTRIBUTING.md`).
3. Write your steps in `workshop/content/*.md` and list them in
   `workshop/config.yaml`.
4. Use the bundled **`educates-workshop-authoring`** skill (in `.claude/skills`)
   for the full authoring reference.
5. Commit, push, and ask an OTM operator to bless your commit SHA.

Thanks for building with OTM!
