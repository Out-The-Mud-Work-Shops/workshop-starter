# Environment Check

Run a command in the terminal to confirm your session is live. Click the
command below to copy it into the terminal:

```terminal:execute
command: echo "Hello from $(whoami) in session $SESSION_NAME"
```

You should see your username and the unique session name.

The editor on the left opens this workshop's files. Try opening the overview:

```editor:open-file
file: ~/exercises/README.md
```

> If the file isn't there yet, that's fine — this starter keeps `setup.d`
> minimal. Add your own exercises under `workshop/setup.d/` to populate the
> session home directory.

Click **Next** to wrap up.
